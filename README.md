## 预期功能列表
- Proportion：单样本比率置信区间、样本量；单样本比率功效、样本量。
- Bland-Altman：Bland-Altman 分析的功效、样本量、Summary、Plot。
- Table：原始数据转四方格表，四方格表相关统计如Se、Sp、NPV、PPV等。
- ROC：单组cut-off、AUC分析；多组比较、回归；绘图。
- Reference-Interval：参数、百分位数、稳健方法。
